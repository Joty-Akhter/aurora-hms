## Provident Fund Management

### Managing EPF Accounts

1. **Navigate to**: HR → Provident Fund → PF Accounts
2. **Create New Account**:
   - Click "Create Account"
   - Enter employee details
   - Enter EPF account number and UAN
   - Set opening date and initial balance
   - Click "Save"

3. **View Account Details**:
   - Click on any account to view details
   - View current balance, contributions, and transactions

### Processing Contributions

1. **Navigate to**: HR → Provident Fund → PF Contributions
2. **Create Contribution**:
   - Select EPF account
   - Enter contribution period (month/year)
   - Enter employee basic salary
   - System calculates employee and employer contributions
   - Review and save

3. **Bulk Processing**:
   - Use payroll integration for automatic contribution processing
   - Contributions are created during payroll run

### Interest Calculations

1. **Navigate to**: HR → Provident Fund → PF Interest
2. **Calculate Interest**:
   - Select financial year
   - Enter interest rate
   - Select accounts or process all
   - Click "Calculate Interest"
   - Review calculations and post

### Withdrawal Management

1. **Navigate to**: HR → Provident Fund → PF Withdrawals
2. **Review Withdrawal Requests**:
   - View pending withdrawal requests from employees
   - Review request details and supporting documents
   - Approve or reject requests
   - Process approved withdrawals

### Nomination Management

1. **Navigate to**: HR → Provident Fund → PF Nominations
2. **View Nominations**:
   - View all nominations for an account
   - Verify nomination details
   - Update if needed

### Compliance Management

1. **Navigate to**: HR → Provident Fund → Advanced → Compliance
2. **Check Compliance**:
   - Select month and year
   - Click "Check Compliance"
   - Review compliance status
   - Address any non-compliance issues

3. **Automate Compliance**:
   - Use "Automate Compliance" to process statutory contributions
   - System automatically calculates and records compliance

## Incentive Management

### Creating Incentive Plans

1. **Navigate to**: HR → Incentives
2. **Create Plan**:
   - Click "Create Incentive Plan"
   - Enter plan details:
     - Plan name and code
     - Plan type (sales, performance, project, retention, referral)
     - Effective dates
     - Eligibility rules (JSON format)
     - Calculation formula
   - Save plan

3. **Configure Eligibility**:
   - Define eligibility rules in JSON format
   - Set department, role, or custom criteria
   - Test eligibility rules

### Managing Sales Targets

1. **Navigate to**: HR → Sales Targets
2. **Create Sales Target**:
   - Select employee
   - Enter target month and year
   - Enter target amount
   - Click "Save"

3. **Update Achievement**:
   - Select sales target
   - Enter achievement amount
   - System calculates achievement percentage
   - Save updates

4. **Configure Target Rules**:
   - Navigate to Sales Target Configuration
   - Create configuration for department/role
   - Set tier structure, thresholds, and caps
   - Define calculation formulas

### Processing Incentives

1. **Navigate to**: HR → Incentives → Payouts
2. **Calculate Incentives**:
   - System automatically calculates based on targets and achievements
   - Review calculated amounts
   - Approve or adjust as needed

3. **Process Payouts**:
   - Select pending payouts
   - Link to payroll run
   - Process payouts
   - Track payment status

### Advanced Incentive Features

1. **Performance Incentives**:
   - Navigate to HR → Incentives → Performance Incentives
   - Calculate individual, team, department, or company-wide incentives
   - Review and approve calculations

2. **Project Incentives**:
   - Navigate to HR → Incentives → Project Incentives
   - Create milestone incentives
   - Calculate completion bonuses
   - Distribute project incentives

3. **Retention & Referral**:
   - Navigate to HR → Incentives → Retention & Referral
   - Create retention bonuses
   - Manage referral program
   - Validate and process referrals

### Dispute Resolution

1. **Navigate to**: HR → Incentives → Disputes
2. **Review Disputes**:
   - View employee disputes
   - Review dispute details
   - Resolve disputes with notes
   - Update incentive calculations if needed

## Sales Target Management

### Setting Monthly Targets

1. **Navigate to**: HR → Sales Targets
2. **Create Targets**:
   - Individual targets: Select employee and set target
   - Team targets: Select team lead and set team target
   - Department targets: Select department and set target

3. **Target Approval**:
   - Review targets before finalizing
   - Approve or request adjustments
   - Communicate targets to employees

### Monitoring Achievement

1. **Navigate to**: HR → Sales Targets → Dashboard
2. **View Achievement**:
   - Real-time achievement tracking
   - Filter by month/year
   - View individual and team performance
   - Monitor achievement percentages

3. **Update Achievement**:
   - Manually update achievement amounts
   - Or integrate with sales system for automatic updates

## Reporting & Analytics

### Provident Fund Reports

1. **Executive Dashboard**:
   - Navigate to HR → Reports → Provident Fund
   - View high-level PF metrics
   - Total balance, participation rate, compliance status

2. **Manager Reports**:
   - Select manager and department
   - View team PF summary
   - Contribution trends

3. **Compliance Reports**:
   - Generate compliance reports
   - Export for audit purposes
   - Track compliance history

### Incentive Reports

1. **Sales Achievement Dashboard**:
   - Navigate to HR → Reports → Incentives
   - View sales achievement metrics
   - Target vs achievement analysis

2. **Individual Reports**:
   - Select employee
   - View incentive history
   - Payout details

3. **Cost Analysis**:
   - Analyze incentive costs
   - ROI analysis
   - Budget planning

### Advanced Analytics

1. **Predictive Analytics**:
   - Navigate to HR → Analytics
   - Predict sales achievement
   - Forecast incentive costs
   - Analyze trends

2. **Custom Reports**:
   - Navigate to HR → Reports → Custom
   - Build custom reports
   - Configure data sources and filters
   - Export in various formats

3. **Scheduled Reports**:
   - Navigate to HR → Reports → Scheduled
   - Create scheduled reports
   - Set frequency and recipients
   - Automate report distribution

## Best Practices

1. **Regular Updates**: Update sales achievements regularly for accurate incentive calculations
2. **Compliance**: Run compliance checks monthly to ensure statutory requirements
3. **Documentation**: Maintain proper documentation for all transactions
4. **Approval Workflows**: Follow approval workflows for withdrawals and incentives
5. **Data Validation**: Verify data before processing bulk operations

## Troubleshooting

### Common Issues

1. **Incentive Not Calculating**:
   - Check if employee is eligible for the plan
   - Verify sales target is set
   - Check calculation period

2. **Contribution Errors**:
   - Verify account is active
   - Check contribution rates
   - Validate payroll integration

3. **Compliance Issues**:
   - Review compliance records
   - Check contribution amounts
   - Verify statutory rates
