## 4. Integration with Payroll and Other Modules – Gaps

**Current state:** `PayrollIntegrationService` integrates EPF and incentives with payroll runs; `PayrollCalculationService` populates `PayrollRun`/`PayrollDetail`/`PayrollComponent` from salary management (assignment + component values) for a period. PayrollDetail has basic/gross/deductions/net; `PayslipDto` exposes component lines, currency and pay frequency. Payroll period lock is enforced via `SalaryService` against closed payroll runs; tax and accounting integration are implemented via salary component semantics and dedicated exports; loan integration and generic integration patterns are still pending.
